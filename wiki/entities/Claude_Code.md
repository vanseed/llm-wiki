# Claude Code

## 定义

Claude Code 是 [[摘要-llm-wiki]] 中列举的 LLM agent 环境之一。它可通过类似 CLAUDE.md 的入口规范理解项目约定，并承担 [[LLM_Wiki]] 的维护工作。

## 在 LLM Wiki 中的角色

Claude Code 代表一种“把 LLM 放进文件系统和代码库式工作区”的使用方式。在 [[Obsidian]] vault 中，它可以依据 [[Wiki_Schema]] 执行 [[Ingest_Workflow]]、[[Query_Workflow]] 和 [[Wiki_Lint]]，把知识库当作可维护的代码库来处理。
